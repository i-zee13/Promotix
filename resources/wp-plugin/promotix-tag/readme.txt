Install the Promotix tracking tag on your WordPress site.

== Description ==

Promotix Tag injects the Promotix tracking script into your WordPress site's `<head>` so you can monitor traffic, clicks, and threats from your Promotix dashboard.

After installing:

1. Open **Settings → Promotix Tag**
2. Paste your **Server URL** and keys from your Promotix dashboard:
   - Domain key
   - Secret key
   - Authentication key
3. Save.

== Installation ==

1. In WordPress Admin, go to **Plugins → Add New → Upload Plugin**
2. Upload the ZIP file and click **Install Now**
3. Click **Activate**
4. Go to **Settings → Promotix Tag** and paste your keys.

== Frequently Asked Questions ==

= Where do I get my keys? =
From your Promotix dashboard in the domain setup screen.

= Does this plugin work with caching? =
If your cache plugin strips tags or defers scripts, exclude the Promotix tag for best accuracy.
